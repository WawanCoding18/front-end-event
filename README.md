WanCourse
A web application built with Next.js that allows users to register and login as Member or Admin, with separate dashboards for each role.

## Features
- User Registration 📝
- User Login 🔑 (Member / Admin)
- Authentication handled with NextAuth.js / JWT 
- Role-based access control:
Member Dashboard – view Dashboard, Transaction, and Setting
Admin Dashboard – view Dashboard, manages Event, Category, Banner, and Transaction WanCourse

## Tech Stack
- Express JS
- Next.JS
- TypeScript
- Tailwind
- MongoDB

## How to Run
1. Clone this repository

   ```bash
   git clone https://github.com/WawanCoding18/front-end-event.git
   cd front-end-event

   ```

2. Install dependencies:
   ```bash
   npm install
   ```
